# Umi-OCR 在线 OCR 插件

本项目为 [Umi-OCR](https://github.com/hiroi-sora/Umi-OCR) 提供了调用**腾讯云 OCR API** 和**百度云 OCR API** 的插件功能，让您可以在 Umi-OCR 中使用云端 OCR 服务。

## ✨ 特性

- 🚀 支持腾讯云和百度云两大平台的多种 OCR 接口
- 🔌 即插即用，安装简单
- 🌍 支持多语言识别（中文、英文、日文、韩文等）
- 📊 支持表格识别、公式识别、手写体识别等专业场景
- ⚙️ 灵活的参数配置，满足不同识别需求

## 📦 安装方法

1. 将 `BaiduOCR` 文件夹和 `TencentOCR` 文件夹复制到 Umi-OCR 的插件目录：
   ```
   UmiOCR-data\plugins\
   ```

2. 重启 Umi-OCR，插件将自动加载

![安装示例](image/Readme/1760349834586.png)

## 🔑 配置步骤

### 1. 申请 API 密钥

参考 [PandaOCRPro 教程](https://www.showdoc.com.cn/PandaOCR/11558472866674591)：

- **腾讯云**：注册[腾讯云账号](https://cloud.tencent.com/)，开通 OCR 服务，在控制台获取 `SecretId` 和 `SecretKey`
- **百度云**：注册[百度智能云账号](https://cloud.baidu.com/)，创建 OCR 应用，获取 `API Key` 和 `Secret Key`

### 2. 填写密钥

在 Umi-OCR 的**全局设置**中填入对应的 API 密钥：

![配置密钥](image/Readme/1760329596317.png)

## 📋 支持的接口

### 腾讯云 OCR (TencentOCR)

支持以下 11 种识别接口：

| 序号 | 接口名称 | 说明 |
|------|----------|------|
| 1 | 通用印刷体识别 | 适用于印刷文档、网络图片等场景 |
| 2 | 通用文字识别（高精度版） | 更高的识别精度，适合要求较高的场景 |
| 3 | 通用印刷体识别（精简版） | 轻量级接口，处理速度快 |
| 4 | 通用印刷体识别（高速版） | 高速处理，适合批量识别 |
| 5 | 公式识别 | 识别数学公式、方程式 |
| 6 | 通用手写体识别 | 识别手写文字 |
| 7 | 表格识别（V1） | 基础表格识别 |
| 8 | 表格识别（V2） | 改进版表格识别 |
| 9 | 表格识别（V3） | 最新版表格识别，精度更高 |
| 10 | 英文识别 | 专门针对英文文本优化 |
| 11 | 广告识别 | 识别广告图片中的文字 |

![腾讯云OCR配置](image/Readme/1760349478493.png)

### 百度云 OCR (BaiduOCR)

支持以下 9 种识别接口：

| 序号 | 接口名称 | 说明 |
|------|----------|------|
| 1 | 通用文字识别（标准版） | 基础文字识别 |
| 2 | 通用文字识别（标准含位置版） | 返回文字位置信息 |
| 3 | 通用文字识别（高精度版） | 更高的识别准确率 |
| 4 | 通用文字识别（高精度含位置版） | 高精度 + 位置信息 |
| 5 | 网络图片文字识别 | 针对网络图片优化 |
| 6 | 表格文字识别 V2 | 表格内容识别 |
| 7 | 数字识别 | 专门识别数字 |
| 8 | 手写文字识别 | 识别手写内容 |
| 9 | 数学公式识别 | 识别数学公式 |

![百度云OCR配置](image/Readme/1760349391483.png)

## 🎯 使用场景

- 📄 **文档数字化**：将纸质文档、图片转换为可编辑文本
- 📊 **表格提取**：从图片中提取表格数据
- ✍️ **手写识别**：识别手写笔记、表单
- 🔢 **公式识别**：识别数学公式、科学公式
- 🌐 **多语言识别**：支持中英日韩等多种语言

## 💰 费用说明

- 腾讯云和百度云的 OCR 服务均提供**免费额度**
- 超出免费额度后将按次计费，具体价格请参考官方定价
- 建议先使用免费额度测试效果
